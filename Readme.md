# VECTORIA

Air Traffic Control Simulator — Unity 6 LTS

## Setup

1. Clona el repo
2. Abre con Unity 6.0.x LTS
3. Escena principal: `Assets/Scenes/RadarScene`

## Stack
- Unity 6 LTS (2D Core)
- C# / .NET Standard 2.1
- Input System 1.7+
- Newtonsoft.Json
- SQLite-net-pcl (Fase 2+)
